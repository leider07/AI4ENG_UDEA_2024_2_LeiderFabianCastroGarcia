### *Descripción del Proyecto*
Este proyecto está orientado a desarrollar un modelo predictivo para la competencia AI4ENG UDEA 2024-2.

### *Archivos en el Repositorio*
- 01 - exploración.ipynb: Notebook para la exploración y análisis preliminar de los datos.
- 02 - preprocesado.ipynb: Notebook para las operaciones de limpieza y preprocesado de los datos.
- 03 - modelo con preprocesado inicial y xgb.ipynb: Notebook con preprocesado relativamente manual y prediccion de la variable objetivo usando xgb.
- 04 - modelo con preprocesado inicial y Random Forest.ipynb: Notebook con preprocesado relativamente manual y prediccion de la variable objetivo usando Random Forest.
- 99 - modelo solución.ipynb: Notebook con preprocesado usando Knnimputer de la librería Sklearn y prediccion de la variale objetivo usando gbc.
